Specification

We consider 2 classes: Dictionaries, and Entries.

The class of Entries is partitioned into 2 non-overlapping subclasses: Definitions, and Words.

We make the following assumptions:

  1. For each Dictionary there is a set of Words that are the words_in it.
  2. For each Dictionary there is some Entry that is the entry_for each word_in it.
  3. For each Dictionary there is at most one entry_for each word_in it.
  4. For each Dictionary there is no word_in it listed in it's own search.
  5. We can search the Definitions of a Word in a Dictionary by successively browsing the entry_for the words that are also Definitions.

Assertion

Here is the assertion that we thought would be true:

Each search of each word_in each Dictionary lists some Word.

Unsatisfied Goal Example

Student0 can grade Assignment0 they are assigned to, which violates the red goal.

  • Student0 is an assistant for Class0.
  • Professor0 is the instructor of Class0.
  • Assignment0 is associated with Class0.
  • Assignment0 is assigned to Student0.

Partial Solution

There is instructor(s) of each class.

There is assistant(s) for each class.

There is student(s) enrolled_in each class.

There is class(es) associated with each assignment.

There is student(s) assigned to each assignment.

For each class, can be instructors.

For each class, can be assistants.

For each class, can be enrolled.

For each assignment, can be associated with it.

For each assignment, can be assigned to it.

Each student enrolled in a class is assigned to every assignment associated with it.

Notes and Goals

This puzzle involves creating a Grading Policy. The Grading Policy involves persons, classes and assignments.

A person is either a student or a professor.

Professors can be instructors of classes.

Students can be assistants for classes.

Students can be enrolled in classes.

Classes can be associated with assignments.

Students can be assigned to assignments.

Instructors and assistants can grade assignments.

There is exactly one instructor of each class.

There is exactly one class associated with each assignment.

There is at least one student assigned to each assignment.

Each student enrolled in a class is assigned to every assignment associated with it.

There is no student who can grade an assignment they are assigned to.

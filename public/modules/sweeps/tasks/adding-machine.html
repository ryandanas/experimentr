Function Description

[integer]adding-machine([integer] sequence)

The adding-machine function takes in a list of integers (sequence) and returns a list of integer sums. Each sum is calculated by adding every number together until a 0 is reached in the sequence. The function stops calculating new sums and immediately returns when encountering two consecutive 0s.
